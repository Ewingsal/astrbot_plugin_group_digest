# AGENTS

本文件用于后续协作开发时统一约束。

## 项目目标

开发 AstrBot 插件：群聊兴趣日报与主动互动插件。

## 当前阶段边界

- 保持 MVP 简洁，先打通主链路：
  - 命令触发 `/group_digest`
  - 昨日报告生成
  - 主动互动文案生成
- 仅修改当前插件仓库，不改 AstrBot 本体。
- 输出形态固定为纯文本。

## 代码约束

- 优先可读性与可维护性，避免过度抽象。
- 不确定的 AstrBot API 通过适配函数隔离，并标注 TODO。
- 纯逻辑必须可被单测覆盖，尽量避免把逻辑写死在事件处理函数里。

## 下一阶段重点

- 将本地 JSON 归档从 MVP 提升到可持续方案（分片或轻量数据库）。
- 完善“昨天”窗口的时区与跨天边界处理。
- 增加真实群聊数据下的统计精度（关键词、成员兴趣摘要）。
- 预留并实现：主动发送到群、定时任务、HTML 转图片。
